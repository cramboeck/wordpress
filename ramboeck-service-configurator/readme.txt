Professioneller Service-Konfigurator für IT-Dienstleister und MSPs.

== Description ==

Ein interaktiver Service-Konfigurator für WordPress, der IT-Dienstleister dabei unterstützt, qualifizierte Leads zu generieren und Angebote zu erstellen.

= Features =

* 3-Schritt-Konfigurator
* 10 vorkonfigurierte IT-Services
* Live-Preisberechnung
* Lead-Management
* E-Mail-Benachrichtigungen
* Vollständig anpassbar

== Installation ==

1. ZIP hochladen via Plugins → Installieren
2. Plugin aktivieren
3. Shortcode `[ramboeck_configurator]` verwenden

== Frequently Asked Questions ==

= Wie verwende ich den Konfigurator? =

Füge den Shortcode `[ramboeck_configurator]` auf einer Seite ein.

= Kann ich die Services anpassen? =

Ja, unter IT Services → Services können alle Services bearbeitet werden.
